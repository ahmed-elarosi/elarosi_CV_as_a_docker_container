# 📄 Ahmed Elarosi – CV as a Docker Container

This repository contains my Curriculum Vitae packaged as a lightweight and secure Docker image.

The container serves a simple web interface where my CV can be accessed in:
- 🇬🇧 English
- 🇩🇪 German
---
## 🚀 Quick Start
**Pull the image:**
```
docker pull ahmedelarosi/ahmed-elarosi-cv:1.0.0
```
**Run the container:**
```
docker run -p 8080:8080 ahmedelarosi/ahmed-elarosi-cv:1.0.0
```
**Open in your browser:** http://localhost:8080
